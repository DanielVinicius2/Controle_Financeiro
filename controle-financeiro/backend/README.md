# 💸 Controle Financeiro — Finflow

Sistema de controle financeiro com Node.js + Supabase.

---

```
backend/
├── index.js          ← Servidor Express + Supabase
├── package.json
└── front/
    ├── index.html    ← Interface web
    ├── style.css     ← Estilos
    └── script.js     ← Lógica do frontend
```

---

## 🚀 Instalação e configuração

### 1. Supabase — Criar tabelas

No **SQL Editor** do Supabase, execute:

```sql
create table categorias (
    id bigint generated by default as identity primary key,
    nome text
);

create table movimentacoes (
    id bigint generated by default as identity primary key,
    descricao text,
    valor numeric,
    tipo text,
    categoria_id bigint references categorias(id)
);
```

### 2. Inserir categorias

```sql
insert into categorias (nome)
values
('Alimentação'),
('Transporte'),
('Salário'),
('Lazer');
```

### 3. Desativar RLS

No **Table Editor** do Supabase:
- `categorias` → RLS → **OFF**
- `movimentacoes` → RLS → **OFF**

### 4. Configurar credenciais

No arquivo `index.js`, substitua:

```js
const supabaseUrl = "SUA_URL_SUPABASE";    // Project Settings → API → Project URL
const supabaseKey = "SUA_CHAVE_SUPABASE";  // Project Settings → API → anon/public
```

### 5. Instalar dependências

```bash
cd backend
npm install
```

### 6. Rodar o servidor

```bash
npm start
```

Acesse: **http://localhost:3000**
